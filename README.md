
## Chess Game

A simple chess game.


### Instructions:

- Clone the repository.
```bash
git clone git@github.com:boolfalse/chess-game.git && cd chess-game
```

- Install the dependencies.
```bash
npm install
```

- Run the application (dev mode with Vite).
```bash
npm run dev
```
